About The Cursed Land

The Cursed Land is a cross-platform Massively Multiplayer Online Role-Playing Game (MMORPG) developed by PopuGames for mobile devices and PC. The game brings together the best of Web 2.0, Web 3.0, and AI gaming, built on MultiversX. As a free-to-play and play-to-earn game with microtransactions, it includes on-chain token utility and NFT integration.
